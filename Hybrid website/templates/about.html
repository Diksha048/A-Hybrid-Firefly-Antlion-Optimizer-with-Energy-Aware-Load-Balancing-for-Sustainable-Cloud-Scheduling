Energy-Efficient Cloud Task Scheduling

Hybrid FAO is an optimization algorithm for scheduling tasks in cloud computing environments. It combines the strengths of the Firefly and Antlion optimization techniques to reduce energy consumption while maintaining high performance.

Technologies

The system is implemented in Python and leverages popular libraries such as NumPy, SciPy, and TensorFlow.
